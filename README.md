# Bundler

A faster-than-light JS bundler

# Installation

```bash
$ yarn add https://github.com/simonhochrein/bundler
```

or

```bash
$ npm install https://github.com/simonhochrein/bundler
```

# Running

```bash
$ bundler ./path/to/files/*.js
```

More options comming soon!
